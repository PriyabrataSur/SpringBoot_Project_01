# SpringBoot_Project_01

## User Management API (CRUD + JWT)  
### Objective:  
Build a backend system to manage users with authentication.  
### Features:  
- Register user  
- Login user (JWT token)  
- Get all users  
- Update user  
- Delete user  
### Endpoints:  
POST /register  
POST /login  
GET /users  
PUT /users/{id}  
DELETE /users/{id}
